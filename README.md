# 🐼 Module Pandas — Manipulation & Analyse de Données

> Module pédagogique complet sur la bibliothèque Pandas, présenté dans le cadre de la formation **P8 Dev Data** à l'Orange Digital Center.

## 📋 Description

Ce projet couvre l'ensemble des concepts fondamentaux de Pandas à travers **7 blocs thématiques**, avec un jeu de données réaliste contenant des données sales à nettoyer.

## 🗂️ Structure du projet

    Module_Pandas/
    ├── notebooks/
    │   └── pandas_presentation.ipynb
    ├── data/

    ├── docs/

## 📚 Contenu des 7 Blocs

| Bloc | Thème |
|------|-------|
| 1 | Introduction à Pandas — Series & DataFrame |
| 2 | Lecture et exploration des données |
| 3 | Sélection et filtrage |
| 4 | Nettoyage des données |
| 5 | Analyse et agrégation |
| 6 | Fusion et concaténation |
| 7 | Pandas et SQL |

## 📊 Jeu de données

Le fichier `promotion_p8.csv` contient des données fictives de 20 apprenants de la promotion P8 avec des imperfections intentionnelles :

- Valeurs manquantes (notes, âge, nom)
- Doublons
- Casse incohérente sur les colonnes Filiere et Statut
